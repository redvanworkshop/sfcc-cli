Install
---

> There are two ways to install SFCC CLI

#### Install via Clone

```bash
cd ~
git clone https://github.com/redvanworkshop/sfcc-cli.git
cd sfcc-cli
npm install -g
```

This method is not currently available, but will be once the project is complete.
